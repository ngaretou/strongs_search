Download the file

Click on the button and your new file will download to your Download folder.

Back up your old appDef file

There are several ways to do the next step, but a good one is to rename your old appDef, copy the new appDef into the folder, and rename the new appDef name the original name.

Enable the new transliteration

If you have not yet added your transliteration into SAB, just be sure to add the language in Appearance/Interface in the Lanugages tab with the same language code.

If you see problems with variables like %s or %book, you can deal with them in TECkit map by adding something like this after the SFM section: 

----------------------------------------
; ----------

; Scripture App Builder (SAB) related 

; ----------

'%,d'                      >       '%,d'

'%book '                   >       '%book '

'%chapter '                >       '%chapter '

'%s'                       >       '%s'

----------------------------------------


If something goes wrong or takes some time to figure out, you can go back to your backed up file.
